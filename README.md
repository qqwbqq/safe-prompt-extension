# Safe Prompt

Safe Prompt 是一个面向设计师、前端开发者、AI 绘图用户和内容创作者的浏览器扩展。

它可以把图片、网页、前端代码和视频关键帧转换成可编辑的提示词，适用于：

- 图片生图提示词提取
- 图片修图提示词提取
- UI / 页面风格拆解
- 前端结构还原提示词
- 音视频 / 分镜提示词生成

## 主要功能

- 右键图片后直接调用插件，只生成图片相关提示词，不会混入前端或视频内容。
- 支持图片模式、页面模式、代码模式、视频模式。
- 支持中文 / English 结果切换。
- 图片模式同时输出：
  - 图像生成提示词
  - 修图提示词
  - 限制词 / 反向提示词
- 页面 / 代码模式可提取：
  - 前端提示词
  - UI 提示词
- 视频模式可提取：
  - 音视频提示词
  - 限制词

## 适合谁用

- 想从参考图中提取高质量提示词的 AI 绘图用户
- 想从网页或代码里反推布局和视觉风格的前端开发者
- 想提取镜头语言、节奏和画面方向的视频创作者

## 使用方法

### 1. 图片模式

适合从参考图里提取提示词，或者把现有图片改写成修图提示词。

步骤：

1. 打开插件
2. 切到“图片”
3. 上传一张或多张图片
4. 点击“生成提示词”
5. 在结果区查看：
   - 图像提示词
   - 修图提示词
   - 限制词

### 2. 右键图片模式

适合网页参考图、设计图、海报、摄影图直接提词。

步骤：

1. 在网页中把鼠标移动到目标图片上
2. 右键
3. 选择 `Safe Prompt`
4. 在弹出的面板里确认标签
5. 点击“生成提示词”

特点：

- 只围绕当前图片生成
- 不会混入前端 / UI / 视频提示词
- 支持中英切换
- 支持直接编辑结果

### 3. 页面模式

适合分析当前网页的结构、视觉语言和组件布局。

步骤：

1. 打开插件
2. 切到“页面”
3. 点击“采集当前页面”
4. 点击“生成提示词”

适合场景：

- 页面拆解
- UI 复刻
- 前端布局分析

### 4. 代码模式

适合把 HTML / CSS / JSX / Vue / Tailwind 代码转成前端与 UI 提示词。

步骤：

1. 打开插件
2. 切到“代码”
3. 粘贴代码
4. 点击“生成提示词”

当前版本会尽量提取：

- 配色
- 容器宽度
- 导航结构
- 按钮样式
- 边框与圆角
- 模块顺序
- 主要布局线索

### 5. 视频模式

适合从本地视频里抽取关键帧并生成音视频提示词。

步骤：

1. 打开插件
2. 切到“视频”
3. 上传本地视频
4. 等待关键帧抽取完成
5. 点击“生成提示词”

## 结果说明

### 图像提示词

适合 Midjourney、Flux、SDXL 等图像生成模型。

### 修图提示词

适合豆包、修图类模型、局部重绘 / 垫图 / 图像编辑类工作流。

### 前端提示词

更偏向结构、布局、组件层级、交互和实现方向。

### UI 提示词

更偏向配色、字体、卡片风格、按钮风格、层级和视觉语言。

### 限制词

用于防止模型输出方向跑偏。

## 常见问题

### 1. 为什么生成速度会慢一点？

因为插件会根据模式分析图片、页面、代码或视频关键帧，再调用模型生成结构化结果。

### 2. 为什么代码模式有时还原不完全一致？

代码模式会优先依据你粘贴的代码来提取配色、布局和结构线索，但如果代码本身不完整，或者关键样式在外部文件中，结果就可能不够完整。

### 3. 视频会不会直接上传原视频？

不会。当前逻辑是先在本地抽取关键帧，再把关键帧发送给模型。

### 4. 可以编辑生成后的提示词吗？

可以。弹窗和右键面板里的结果都支持直接修改。
