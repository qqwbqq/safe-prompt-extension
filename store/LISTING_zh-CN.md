# Prompt Crafter 商店文案

## 标题

Prompt Crafter - 图片、页面、视频转提示词

## 简短描述

把图片、前端页面代码或视频关键帧转换成高质量 AI 提示词，适用于图像生成、UI 复刻、分镜拆解等场景。

## 详细描述

Prompt Crafter 是一个面向设计师、前端开发者、内容创作者和 AI 工作流用户的浏览器扩展。它可以把多种输入内容快速转换成结构化、高可用的提示词。

支持能力：

- 图片转提示词：上传一张或多张图片，自动提炼风格、构图、材质、光线、镜头和氛围
- 页面转提示词：采集当前页面的结构、样式和可见文本，用于生成 UI 复刻提示词或设计拆解提示词
- 代码转提示词：粘贴 HTML、CSS、JSX、Vue、Tailwind 等代码片段，生成适合设计还原或组件重建的提示词
- 视频转提示词：从本地视频抽取关键帧，生成风格分析、镜头语言和分镜提示词

输出内容包含：

- 核心提示词
- 扩展提示词
- 反向提示词 / 限制项
- 结构化标签

适用场景：

- Midjourney、Flux、Stable Diffusion 等图像生成工作流
- UI 页面还原和视觉风格拆解
- 视频创意提炼和分镜规划
- 前端页面设计语言归纳

隐私说明：

- 图片和视频先在本地处理，视频默认只抽取关键帧
- 只有在用户主动点击生成时，内容才会发送到你配置的 AI 接口
- 当前页面只有在用户点击“采集当前页面”时才会被读取

## 单行卖点

从视觉内容和前端代码中提炼可直接使用的专业提示词。
