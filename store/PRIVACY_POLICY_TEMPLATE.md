# Prompt Crafter 隐私政策模板

最后更新日期：2026-03-31

## 1. 我们收集什么

Prompt Crafter 仅在用户主动操作时处理以下数据：

- 用户上传的图片
- 用户上传的视频所抽取的关键帧
- 用户主动粘贴的前端页面代码
- 用户主动点击采集的当前页面结构信息
- 用户在设置页填写的 API 地址、模型名和 API Key

## 2. 我们如何使用这些数据

这些数据仅用于完成用户发起的提示词生成请求，包括：

- 分析图片内容并生成提示词
- 分析页面结构或代码并生成提示词
- 分析视频关键帧并生成提示词
- 将请求发送到用户自己配置的 AI 接口

## 3. 数据传输

只有在用户点击“生成提示词”时，相关输入数据才会被发送到用户在设置页中配置的 AI 服务接口。

如果用户没有主动触发生成操作，扩展不会自动上传图片、视频、页面内容或代码。

## 4. 本地存储

扩展会在浏览器本地存储以下配置项：

- API 地址
- 模型名
- API Key
- 默认语言与偏好设置

这些信息用于后续请求，不会自动共享给其他第三方服务，除非用户主动配置并使用相应的 AI 接口。

## 5. 页面访问

扩展不会持续监控网页内容。只有当用户点击“采集当前页面”时，扩展才会读取当前活动标签页的页面结构信息。
